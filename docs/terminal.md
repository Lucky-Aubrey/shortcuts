# üêß Terminal Shortcuts
---
Cursor Movement:
| Hotkey/Combo                          | Description                             |
|--------------------------------------|-----------------------------------------|
|Ctrl + a | Go to start of line|
|Ctrl + e | Go to end of line|
|Ctrl + f | Move forward one character|
|Ctrl + b | Move backward one character|
|Alt + f | Move forward one word|
|Alt + b | Move backward one word |
---
Text Editing:
| Hotkey/Combo                          | Description                             |
|--------------------------------------|-----------------------------------------|
|Ctrl + u | Delete from cursor to start of line|
|Ctrl + k | Delete from cursor to end of line|
|Ctrl + w | Delete word before cursor|
|Ctrl + y | Paste the last deleted text|
|Ctrl + t | Swap the character under cursor with the previous one |
---
Command History:
| Hotkey/Combo                          | Description                             |
|--------------------------------------|-----------------------------------------|
|Ctrl + p | Previous command in history|
|Ctrl + n | Next command in history|
|Ctrl + r | Search backward through command history|
|Ctrl + g | Exit history search mode |
---
Process Control:
| Hotkey/Combo                          | Description                             |
|--------------------------------------|-----------------------------------------|
|Ctrl + c | Terminate the current running process|
|Ctrl + z | Suspend the current process (send to background)|
|Ctrl + d | Exit the shell (EOF) or delete the character under cursor |
---
Screen Control:
| Hotkey/Combo                          | Description                             |
|--------------------------------------|-----------------------------------------|
|Ctrl + l | Clear the screen (same as clear)|
|Ctrl + s | Pause terminal output|
|Ctrl + q | Resume terminal output |
